Thesis Fluid Simulation Using Smoothed Particle Hydrodynamics (SPH)

Abstract

Smoothed Particle Hydrodynamics (SPH) is a widely used method of simulating fluids in computer graphics. This paper explains this particular method, mentions previous research that has been done one it, as well as comparing it to other methods. Furthermore,an implementation of SPH is analysed. The implementation was done in C++ and the resulting simulation was imported to Houdini.
